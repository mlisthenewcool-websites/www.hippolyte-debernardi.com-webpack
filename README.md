# Tutorial

## React.js/Typescript/Webpack without create-react-app

* https://grant-bartlett.com/blog/react-typescript-and-webpack-starter-project/

```
yarn init

yarn add react react-dom react-helmet-async react-markdown react-router-dom

yarn add -D typescript @types/node @types/react @types/react-dom @types/react-router-dom

yarn add -D webpack webpack-cli webpack-dev-server awesome-typescript-loader html-webpack-plugin


```
